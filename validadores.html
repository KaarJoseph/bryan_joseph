ESTÁNDARES HTML

La estructura de una página web se compone principalmente de tres elementos: HTML, CSS y JavaScript (JS). Cada uno de estos elementos cumple una función específica en la construcción de una página web y su diseño.

HTML es el lenguaje de marcado utilizado para definir la estructura y el contenido de una página web. Es el lenguaje base de todas las páginas web y se utiliza para definir los elementos como encabezados, párrafos, listas, imágenes, enlaces y más. JavaScript es un lenguaje de programación utilizado para agregar interactividad y dinamismo a una página web. Se utiliza para crear efectos especiales, validar formularios, crear animaciones, actualizar contenido dinámicamente y más.
